#### て型的变化
* ぐく　-> + いて
* うつる -> + 促音って
* むぶる　-> + んで
* すして -> + して

注意：以て结尾的次也可以变成て型（てして）


#### て型的使用
1. 连接动作
2. 原因
3. 顺接
